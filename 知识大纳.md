## 第1章 三角函数

### §1.1 三角函数的概念

**核心：角、弧度制、三角函数定义 + 性质**

* **角与弧度**

  * 度：(360^\circ) 为一周；弧度：以半径为弧长的圆心角为 (1\text{ rad})。
  * 换算：(\pi \text{ rad} = 180^\circ)，弧长公式：(s=R\theta)（(\theta) 用弧度）。
* **任意角与象限角**

  * 终边落在哪个象限 → 象限角；同终边角：相差 (2k\pi)。
* **三角函数的单位圆定义**

  * 单位圆上点 (P(x,y)) 对应角 (\alpha)：(\sin\alpha=y,\ \cos\alpha=x,\ \tan\alpha=\dfrac{y}{x})。
  * 定义域：(\sin,\cos) 定义在 (\mathbb R)，(\tan) 在 (\mathbb R\setminus{\frac{\pi}{2}+k\pi})。
* **同角三角函数关系**

  * ( \sin^2\alpha+\cos^2\alpha=1)；
  * (\tan\alpha=\dfrac{\sin\alpha}{\cos\alpha})，(\cot\alpha=\dfrac{1}{\tan\alpha}) 等。
* **符号、奇偶、周期、单调**

  * 记“**ASTC**”：I 象限全正，II 仅 sin 正，III 仅 tan 正，IV 仅 cos 正。
  * 奇偶：(\sin(-x)=-\sin x)（奇），(\cos(-x)=\cos x)（偶），(\tan(-x)=-\tan x)。
  * 周期：(\sin,\cos) 周期 (2\pi)，(\tan) 周期 (\pi)。
  * 在基本区间上的增减性（后面跟函数、极限衔接）。

### §1.2 两角和与差的三角函数

**核心：和差公式 + 倍角（含万能公式）**

* **和差公式**

  * (\sin(\alpha\pm\beta)=\sin\alpha\cos\beta\pm\cos\alpha\sin\beta)；
  * (\cos(\alpha\pm\beta)=\cos\alpha\cos\beta\mp\sin\alpha\sin\beta)；
  * (\tan(\alpha\pm\beta)=\dfrac{\tan\alpha\pm\tan\beta}{1\mp\tan\alpha\tan\beta})。
* **倍角公式**

  * (\sin2\alpha=2\sin\alpha\cos\alpha)；
  * (\cos2\alpha=\cos^2\alpha-\sin^2\alpha=2\cos^2\alpha-1=1-2\sin^2\alpha)；
  * (\tan2\alpha=\dfrac{2\tan\alpha}{1-\tan^2\alpha})。
* **万能公式（用 (\tan\frac{x}{2}) 表示）**

  * 设 (t=\tan\dfrac{x}{2})：
    (\sin x=\dfrac{2t}{1+t^2},\quad \cos x=\dfrac{1-t^2}{1+t^2},\quad \tan x=\dfrac{2t}{1-t^2})。
* **典型用途**

  * 计算特殊角（如 (15^\circ)、(75^\circ)）；
  * 化简/证明三角恒等式；
  * 为三角方程求解做准备。

### §1.3 三角函数的积化和差与和差化积

**核心：积 ⇄ 和差 的变形工具**

* **积化和差**

  * (\sin\alpha\sin\beta=\dfrac{\cos(\alpha-\beta)-\cos(\alpha+\beta)}{2})；
  * (\cos\alpha\cos\beta=\dfrac{\cos(\alpha-\beta)+\cos(\alpha+\beta)}{2})；
  * (\sin\alpha\cos\beta=\dfrac{\sin(\alpha+\beta)+\sin(\alpha-\beta)}{2})。
* **和差化积**

  * (\sin\alpha+\sin\beta=2\sin\dfrac{\alpha+\beta}{2}\cos\dfrac{\alpha-\beta}{2}) 等类似三式。
* **用法**

  * 求和/求积类三角式化简；
  * 为以后积分中处理 (\sin ax\cos bx) 做铺垫。

## 第2章 代数与方程

### §2.1 代数式及其运算

**核心：代数式、因式分解、有理式运算**

* 代数式：含未知数和运算的式子，区分**整式**与**分式**。
* 运算：合并同类项、去括号、乘除运算。
* 因式分解常用方法：

  * 提公因式；
  * 公式：平方差、完全平方、立方和差等；
  * 分组分解、十字相乘。
* 有理式运算：

  * 通分、约分、分式加减乘除；
  * 注意分母不为 0 的限制。

### §2.2 一元二次方程的性质

**核心：判别式、求根公式、根与系数**

* 标准形式：(ax^2+bx+c=0\ (a\neq0))。
* 判别式：(\Delta=b^2-4ac)，判断实根个数。
* 求根公式：(\displaystyle x=\dfrac{-b\pm\sqrt{\Delta}}{2a})。
* 根与系数关系（Vieta）

  * 若根为 (x_1,x_2)：
    [
    x_1+x_2=-\frac{b}{a},\quad x_1x_2=\frac{c}{a}
    ]
* 图像视角：抛物线与 (x) 轴交点 ↔ 方程实根。

### §2.3 解一元代数方程

**核心：整式/分式/含根号等一元方程的通法**

* 基本步骤：

  1. 去分母、去根号（注意平方会引入增根）；
  2. 整理成多项式方程；
  3. 因式分解或用求根公式；
  4. 回代检查增根。
* 常见类型：

  * 分式方程、无理方程（含根号）、带参数的一元二次方程。

## 第3章 平面几何

### §3.1 三角形

**核心：三角形性质 + 常用定理**

* 内角和 (180^\circ)，外角 = 两个不相邻内角和。
* 特殊三角形：等腰、等边、直角三角形的性质。
* 全等三角形判定：SSS, SAS, ASA, AAS, RHS。
* 面积公式：(S=\frac12ah_a=\frac12ab\sin C)。
* 结合三角函数：边角关系、正弦/余弦定理（向高数矢量推广）。

### §3.2 四边形

**核心：平行四边形族的性质**

* 平行四边形：对边平行且相等，对角相等，对角线互相平分。
* 矩形：四角为直角、对角线相等。
* 菱形：四边相等、对角线垂直且平分角。
* 正方形：同时具备矩形和菱形的性质。

### §3.3 圆

**核心：圆心角、弧、弦、切线与内接角**

* 同弧的圆心角与内接角关系：圆心角 = 2×内接角。
* 弦心距定理：距圆心越近的弦越长。
* 切线性质：切线 ⟂ 半径；切线段定理等。

### §3.4 相似形

**核心：相似三角形与比例**

* 相似判定：AA、SAS（夹角相等）、SSS（成比例）。
* 相似比 k：

  * 周长比 = k；面积比 = (k^2)。
* 应用：几何测量、间接求长、面积计算。

## 第4章 反三角函数

### §4.1 反函数

**核心：一一对应 + 关于 (y=x) 的对称**

* 反函数定义：(y=f(x)) 可逆 ⇔ 在区间上单调，且每个 y 对应唯一 x。
* 图像：反函数是原函数图像关于直线 (y=x) 的对称。
* 求反函数：交换 x,y → 解出 y。

### §4.2 反三角函数

**核心：截取区间定义反函数 + 常见值**

* 定义域/值域：

  * (\arcsin x: x\in[-1,1],\ y\in[-\frac{\pi}{2},\frac{\pi}{2}])；
  * (\arccos x: x\in[-1,1],\ y\in[0,\pi])；
  * (\arctan x: x\in\mathbb R,\ y\in(-\frac{\pi}{2},\frac{\pi}{2}))。
* 关系：

  * (\sin(\arcsin x)=x)、(\cos(\arccos x)=x)、(\tan(\arctan x)=x)；
  * 但 (\arcsin(\sin x)) 需要考虑把 x 化到主值区间。
* 常见计算：(\arcsin\frac{\sqrt3}{2},\arccos(-\frac12)) 等。

### §4.3 三角方程

**核心：基本方程的通解与区间解**

* 基本方程：

  * (\sin x = a)、(\cos x = a)、(\tan x = a) 的通解公式；
* 利用和差、倍角、万能公式化简后再解；
* 在指定区间（如 ([0,2\pi))）内列出所有解。

## 第5章 排列与组合

### §5.1 分类计数原理与分步计数原理

* 分类加法原理：互斥情况求和。
* 分步乘法原理：独立步骤求积。
* 用于搭建计数模型，是后面排列组合的根。

### §5.2 排列

* 定义：从 n 个不同元素中取 r 个排成一列。
* 公式：(A_n^r=\dfrac{n!}{(n-r)!})，全排列 (A_n^n=n!)。
* 特殊限制：相邻、分组、圆排列等变形。

### §5.3 组合

* 定义：只看选哪些元素，不看顺序。
* 公式：(C_n^r=\dfrac{n!}{r!(n-r)!})，对称性 (C_n^r=C_n^{n-r})。
* 典型题：选人、分组、至多/至少类问题。

### §5.4 二项式定理

* 展开：((a+b)^n=\sum_{k=0}^{n}C_n^k a^{n-k}b^k)。
* 重点：指定项系数、系数和、带绝对值的展开等。

### §5.5 数学归纳法

* 步骤：

  1. 证明 (n=1) 成立；
  2. 假设 (n=k) 成立，推出 (n=k+1) 成立。
* 用于：证明公式、不等式、整除性等。

## 第6章 复数

### §6.1 复数的概念

* 形式：(z=a+bi)，(i^2=-1)；
* 实部 (\Re z=a)，虚部 (\Im z=b)；
* 复平面：点 ((a,b)) ↔ 复数 (a+bi)。

### §6.2 复数的运算

* 加减：实部对实部、虚部对虚部。
* 乘法：((a+bi)(c+di)=(ac-bd)+(ad+bc)i)。
* 共轭：(\overline{a+bi}=a-bi)，(|z|=\sqrt{a^2+b^2})。
* 除法：乘以分母的共轭化成实数分母。

### §6.3 数系的扩充

* 自然数 → 整数 → 有理数 → 实数 → 复数；
* 在复数范围内，任意一元二次方程都有根。

### §6.4 复数与平面向量、三角函数的联系

* 向量视角：模 (|z|) = 长度，辐角 (\arg z) = 与 x 轴夹角 (\theta)。
* 三角形式：(z=r(\cos\theta+i\sin\theta))。

### §6.5 复数的指数形式

* 欧拉公式：(\mathrm e^{i\theta}=\cos\theta+i\sin\theta)。
* 指数形式：(z=r\mathrm e^{i\theta})。
* 乘除：模相乘/相除，辐角相加/相减。
* 为大学里的傅里叶分析、信号处理打基础。

## 第7章 参数方程与极坐标方程

### §7.1 参数方程

* 一条曲线由参数 (t) 描述：
  [
  \begin{cases}
  x=x(t)\
  y=y(t)
  \end{cases}
  ]
* 典型：圆、椭圆、直线的参数表示。
* 表示运动轨迹、便于处理复杂曲线，是以后学习曲线积分的准备。

### §7.2 极坐标方程

* 极坐标：点由 ((r,\theta)) 表示，
  [
  x=r\cos\theta,\ y=r\sin\theta
  ]
* 常见曲线：

  * 以原点为圆心的圆：(r=a)；
  * 过原点直线：(\theta=\theta_0)；
  * 一些特殊“花瓣”“螺线”等曲线：(r=f(\theta))。
* 与直角坐标互化：为多元函数、物理中的极坐标应用铺路。
