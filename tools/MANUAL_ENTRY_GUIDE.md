# 人工录入快速指南

## 🎯 目标：高效、快速、准确录入题目

## 方法 1: 使用网页工具（最推荐 ⭐⭐⭐⭐⭐）

### 优势
- ✅ **最快速度**：可视化界面，无需手写 JSON
- ✅ **实时预览**：立即看到题目效果
- ✅ **格式保证**：自动生成正确格式的 JSON
- ✅ **错误率低**：表单验证，减少格式错误

### 使用步骤

1. **打开工具**
   ```bash
   # 在浏览器中打开
   open tools/manual_entry_helper.html
   # 或者直接双击文件
   ```

2. **填写表单**
   - 选择主题和难度
   - 输入题目（支持 LaTeX 快捷按钮）
   - 输入 4 个选项
   - 选择正确答案（点击单选按钮）
   - 输入解答过程
   - 添加标签（可选）

3. **生成和复制**
   - 点击"生成 JSON"
   - 点击"复制 JSON"
   - 打开 `assets/data/problems.json`
   - 找到最后一个题目的 ID（如 p10）
   - 将复制的 JSON 中的 `"id": "p?"` 改为 `"id": "p11"`
   - 粘贴到 JSON 数组中

4. **验证**
   ```bash
   python tools/validate_problems.py
   ```

### 效率提升技巧

- **使用快捷按钮**：点击 LaTeX 快捷按钮快速插入常用公式
- **批量录入**：录入完一道题后，点击"清空表单"继续下一题
- **预览功能**：使用"预览题目"检查显示效果
- **模板复用**：相似题目可以复制 JSON，只修改不同部分

## 方法 2: 使用 AI 辅助生成（推荐 ⭐⭐⭐⭐）

### 优势
- ✅ **速度快**：AI 生成题目内容
- ✅ **质量高**：AI 生成的题目格式规范
- ✅ **只需校对**：人工只需检查和微调

### 使用步骤

1. **打开提示词模板**
   ```bash
   open tools/ai_prompt_template.md
   ```

2. **复制提示词到 AI 工具**
   - 智谱AI: https://open.bigmodel.cn/
   - 通义千问: https://tongyi.aliyun.com/
   - 文心一言: https://yiyan.baidu.com/

3. **修改参数**
   - 修改主题（如"导数基础"）
   - 修改难度（如"基础"）
   - 修改数量（如 10 道）

4. **获取 JSON**
   - 复制 AI 生成的 JSON
   - 使用网页工具验证格式
   - 或直接添加到 problems.json

5. **批量处理**
   - 一次生成 10-20 道题目
   - 使用验证工具检查
   - 批量添加到题库

## 方法 3: 直接编辑 JSON（适合熟悉格式的用户 ⭐⭐⭐）

### 优势
- ✅ **完全控制**：直接编辑，灵活性高
- ✅ **批量操作**：可以一次添加多道题目

### 使用步骤

1. **打开文件**
   ```bash
   code assets/data/problems.json
   ```

2. **参考现有格式**
   ```json
   {
     "id": "p11",
     "topic": "导数基础",
     "difficulty": "基础",
     "question": "求函数 $f(x) = x^2 + 3x + 2$ 的导数。",
     "answer": "A",
     "options": [
       "$2x + 3$",
       "$x + 3$",
       "$2x^2 + 3$",
       "$x^2 + 3$"
     ],
     "solution": "使用基本求导法则：\\n$f'(x) = 2x + 3$",
     "tags": ["导数", "多项式"]
   }
   ```

3. **注意事项**
   - ID 必须唯一且递增
   - LaTeX 公式用 `$...$` 包裹
   - 正确答案必须在 options 第一个位置，answer 为 "A"
   - 换行使用 `\n`
   - JSON 格式必须正确（逗号、引号等）

4. **验证**
   ```bash
   python tools/validate_problems.py
   ```

## ⚠️ 常见错误

1. **LaTeX 格式错误**
   - ❌ 忘记用 `$` 包裹公式
   - ✅ 所有数学公式用 `$...$` 包裹

2. **答案位置错误**
   - ❌ 正确答案不在第一个位置
   - ✅ 正确答案必须在 options[0]，answer 为 "A"

3. **ID 重复或跳跃**
   - ❌ ID 不连续或重复
   - ✅ ID 必须从 p1 开始，连续递增

4. **JSON 格式错误**
   - ❌ 缺少逗号、引号等
   - ✅ 使用验证工具检查
