### 一、单项选择题 (本大题共 5 小题, 每小题 3 分, 共 15 分, 每小题只有一项符合题目要求)

**1. $\lim_{x \to 0} \frac{\tan 6x}{2x} =$**
A. 1
B. 2
C. 3
D. 4
* **【答案】** C
* **【解析】** $\lim_{x \to 0} \frac{\tan 6x}{2x} = \lim_{x \to 0} \frac{6x}{2x} = 3$ (利用等价无穷小 $\tan 6x \sim 6x$ )。

**2. 点 $x=3$ 是 $f(x) = \frac{x^2 - x - 6}{x-3}$ 的**
A. 连续点
B. 可去间断点
C. 无穷间断点
D. 跳跃间断点
* **【答案】** B
* **【解析】** $f(x)$ 在 $x=3$ 处无定义。但 $\lim_{x \to 3} f(x) = \lim_{x \to 3} \frac{(x-3)(x+2)}{x-3} = \lim_{x \to 3} (x+2) = 5$。极限存在，故 $x=3$ 是 $f(x)$ 的可去间断点。

**3. 设 $F(x)$ 是 $f(x)$ 的一个原函数，$C$ 为任意常数，则下列正确的是**
A. $\int F(x) dx = f(x)$
B. $F'(x) = f(x) + C$
C. $f'(x) = F(x) + C$
D. $\int f(x) dx = F(x) + C$
* **【答案】** D
* **【解析】** 根据不定积分的定义，$\int f(x) dx = F(x) + C$。

**4. 设常级数 $\sum_{n=1}^\infty u_n$ 收敛，则下列收敛的是**
A. $\sum_{n=1}^\infty (u_n + \frac{1}{n})$
B. $\sum_{n=1}^\infty (u_n + \frac{1}{2^n})$
C. $\sum_{n=1}^\infty (u_n + \frac{1}{3^n})$
D. $\sum_{n=1}^\infty (u_n - \frac{1}{\sqrt{n}})$
* **【答案】** C
* **【解析】** $\sum_{n=1}^\infty u_n$ 收敛。
    * 选项 A 和 D 包含发散级数 $\sum \frac{1}{n}$ 和 $\sum \frac{1}{\sqrt{n}}$，故发散。
    * 选项 B 和 C 包含收敛级数 $\sum \frac{1}{2^n}$ 和 $\sum \frac{1}{3^n}$ (等比级数)，收敛级数之和仍收敛。根据参考答案，选择 C。

**5. 设 $F(x) = \int_0^{x^2} \sin t^2 dt$， $g(x) = 3x^6 + 4x^5$， 当 $x \to 0$ 时，**
A. $F(x)$ 是比 $g(x)$ 低阶的无穷小
B. $F(x)$ 是比 $g(x)$ 高阶的无穷小
C. $F(x)$ 与 $g(x)$ 是等价无穷小
D. $F(x)$ 与 $g(x)$ 是同阶非等价无穷小
* **【答案】** B
* **【解析】** 采用洛必达法则和等价无穷小代换 $\sin t^2 \sim t^2$:
    $$\lim_{x \to 0} \frac{F(x)}{g(x)} = \lim_{x \to 0} \frac{\int_0^{x^2} \sin t^2 dt}{3x^6 + 4x^5} \sim \lim_{x \to 0} \frac{\int_0^{x^2} t^2 dt}{4x^5} = \lim_{x \to 0} \frac{[\frac{1}{3}t^3]_0^{x^2}}{4x^5}$$
    $$= \lim_{x \to 0} \frac{\frac{1}{3} (x^2)^3}{4x^5} = \lim_{x \to 0} \frac{x^6}{12x^5} = \lim_{x \to 0} \frac{x}{12} = 0$$
    极限为 $0$，故 $F(x)$ 是比 $g(x)$ **高阶的无穷小**。

### 二、填空题 (本大题共 5 小题, 每小题 3 分, 共 15 分)

**6. $\begin{cases} x = 2t^3 + 3 \\ y = t^2 - 4 \end{cases}$，在 $t=1$ 相应的点处的切线斜率为 \_\_\_\_\_\_。**
* **【答案】** $\frac{1}{3}$
* **【解析】** $\frac{dy}{dx} = \frac{dy/dt}{dx/dt} = \frac{2t}{6t^2} = \frac{1}{3t}$。 $\frac{dy}{dx}|_{t=1} = \frac{1}{3(1)} = \frac{1}{3}$。

**7. $z = x^2 y$ 的全微分 $dz = $ \_\_\_\_\_\_。**
* **【答案】** $2xy dx + x^2 dy$
* **【解析】** $dz = \frac{\partial z}{\partial x} dx + \frac{\partial z}{\partial y} dy = 2xy dx + x^2 dy$。

**8. 微分方程 $\frac{dy}{dx} = y+2$ 满足初值 $\left. y \right|_{x=0} = -1$ 的特解是 \_\_\_\_\_\_。**
* **【答案】** $e^x - 2$
* **【解析】** $\frac{dy}{y+2} = dx$，积分得 $\ln|y+2| = x + C_1$，即 $y+2 = C e^x$。代入初值 $x=0, y=-1$：$-1+2 = C e^0$，得 $C=1$。特解为 $y = e^x - 2$。

**9. 已知 $D = \{(x,y) | 0 \le x \le 1, 0 \le y \le 3-x \}$，则 $\iint_D d\sigma =$ \_\_\_\_\_\_。**
* **【答案】** $\frac{5}{2}$
* **【解析】** $\iint_D d\sigma$ 即为区域 $D$ 的面积 $S_D$。
    $$S_D = \int_0^1 dx \int_0^{3-x} dy = \int_0^1 (3-x) dx = [3x - \frac{1}{2}x^2]_0^1 = 3 - \frac{1}{2} = \frac{5}{2}$$

**10. 连续函数 $f(x)$ 满足 $\int_0^{2x+1} f(t) dt = -2x^3 - \frac{1}{4}$，则 $f(3) = $ \_\_\_\_\_\_。**
* **【答案】** $-3$
* **【解析】** 对等式两边关于 $x$ 求导： $f(2x+1) \cdot (2x+1)' = -6x^2$。
    $2 f(2x+1) = -6x^2$，即 $f(2x+1) = -3x^2$。
    令 $2x+1 = 3$，得 $x=1$。
    $f(3) = -3(1)^2 = -3$。

### 三、计算题 (本大题共 8 小题, 每小题 6 分, 共 48 分)

**11. 求极限 $\lim_{x \to +\infty} (\sqrt{x^2+3} - x)$ 的值。**
* **【答案】** $0$
* **【解析】** (有理化)
    $$\lim_{x \to +\infty} (\sqrt{x^2+3} - x) = \lim_{x \to +\infty} \frac{x^2+3 - x^2}{\sqrt{x^2+3} + x}$$
    $$= \lim_{x \to +\infty} \frac{3}{\sqrt{x^2+3} + x} = \lim_{x \to +\infty} \frac{3}{x(\sqrt{1+\frac{3}{x^2}} + 1)} = \frac{3}{+\infty \cdot 2} = 0$$

**12. 已知 $y = 2^x + x^x$ ($x>0$), 求 $\frac{dy}{dx}$。**
* **【答案】** $2^x \ln 2 + x^x (\ln x + 1)$
* **【解析】** $(2^x)' = 2^x \ln 2$。对于 $x^x$，取对数 $\ln (x^x) = x \ln x$，求导得 $\frac{1}{x^x} (x^x)' = \ln x + 1$，故 $(x^x)' = x^x (\ln x + 1)$。
    $$\frac{dy}{dx} = (2^x)' + (x^x)' = 2^x \ln 2 + x^x (\ln x + 1)$$

**13. 求不定积分 $\int (x+5) \cos 3x dx$。**
* **【答案】** $\frac{1}{3} (x+5) \sin 3x + \frac{1}{9} \cos 3x + C$
* **【解析】** (分部积分法 $\int u dv = uv - \int v du$)
    $$\int (x+5) \cos 3x dx = \frac{1}{3} (x+5) \sin 3x - \int \frac{1}{3} \sin 3x dx$$
    $$= \frac{1}{3} (x+5) \sin 3x - \frac{1}{3} (-\frac{1}{3} \cos 3x) + C$$
    $$= \frac{1}{3} (x+5) \sin 3x + \frac{1}{9} \cos 3x + C$$

**14. 求定积分 $\int_{-2}^2 \frac{x^{2021} + |x|}{x^2+1} dx$。**
* **【答案】** $\ln 5$
* **【解析】** $\frac{x^{2021}}{x^2+1}$ 为奇函数，在对称区间 $[-2, 2]$ 上的积分为 $0$。 $\frac{|x|}{x^2+1}$ 为偶函数。
    $$\int_{-2}^2 \frac{x^{2021} + |x|}{x^2+1} dx = \int_{-2}^2 \frac{x^{2021}}{x^2+1} dx + \int_{-2}^2 \frac{|x|}{x^2+1} dx$$
    $$= 0 + 2 \int_0^2 \frac{x}{x^2+1} dx$$
    令 $u = x^2+1$，则 $du = 2x dx$。
    $$= 2 \int_1^5 \frac{1}{u} \cdot \frac{1}{2} du = [\ln|u|]_1^5 = \ln 5 - \ln 1 = \ln 5$$

**15. 已知 $z = z(x,y)$ 是方程 $e^z - xz = 1$ 确定的隐函数，求 $\frac{\partial z}{\partial x} + \frac{\partial z}{\partial y}$。**
* **【答案】** $\frac{z}{e^z - x}$
* **【解析】** 方程两边对 $y$ 求偏导： $e^z \frac{\partial z}{\partial y} - x \frac{\partial z}{\partial y} = 0$，得 $\frac{\partial z}{\partial y} = 0$。
    方程两边对 $x$ 求偏导： $e^z \frac{\partial z}{\partial x} - (z + x \frac{\partial z}{\partial x}) = 0$，得 $(e^z - x) \frac{\partial z}{\partial x} = z$，即 $\frac{\partial z}{\partial x} = \frac{z}{e^z - x}$。
    $$\frac{\partial z}{\partial x} + \frac{\partial z}{\partial y} = \frac{z}{e^z - x} + 0 = \frac{z}{e^z - x}$$

**16. 已知平面区域 $D$ 为 $x^2 + y^2 \le 4$ 在第一象限的部分，求 $\iint_D e^{x^2+y^2} d\sigma$。**
*【注：本题附有区域 $D$ 的图示，请参考原试卷】*
* **【答案】** $\frac{\pi}{4} (e^4 - 1)$
* **【解析】** (极坐标变换) 区域 $D$: $0 \le r \le 2, 0 \le \theta \le \frac{\pi}{2}$。
    $$\iint_D e^{x^2+y^2} d\sigma = \int_0^{\pi/2} d\theta \int_0^2 e^{r^2} r dr$$
    内层积分：$\int_0^2 e^{r^2} r dr = \frac{1}{2} \int_0^4 e^u du = \frac{1}{2} [e^u]_0^4 = \frac{1}{2} (e^4 - 1)$。
    外层积分：$\int_0^{\pi/2} \frac{1}{2} (e^4 - 1) d\theta = \frac{1}{2} (e^4 - 1) [\theta]_0^{\pi/2} = \frac{\pi}{4} (e^4 - 1)$。

**17. 判断 $\sum_{n=1}^\infty \left(\frac{n}{2n+1}\right)^n$ 的收敛性。**
* **【答案】** 收敛
* **【解析】** (根值审敛法)
    $$L = \lim_{n \to \infty} \sqrt[n]{|a_n|} = \lim_{n \to \infty} \frac{n}{2n+1} = \frac{1}{2}$$
    因为 $L = \frac{1}{2} < 1$，故级数收敛。

**18. 设 $f(x) = \begin{cases} x^2, & x \le 2 \\ 6-x, & x > 2 \end{cases}$，$F(x) = \int_0^x f(t) dt$。求 $F(x)$ 在点 $x=2$ 处的连续性。**
* **【答案】** 连续
* **【解析】**
    当 $x \le 2$ 时，$F(x) = \int_0^x t^2 dt = \frac{1}{3} x^3$。
    当 $x > 2$ 时，$F(x) = \int_0^2 t^2 dt + \int_2^x (6-t) dt = \frac{8}{3} + [6t - \frac{1}{2} t^2]_2^x = \frac{8}{3} + (6x - \frac{1}{2} x^2) - (12 - 2) = 6x - \frac{1}{2} x^2 - \frac{22}{3}$。
    检查 $x=2$ 处的极限和函数值：
    * $\lim_{x \to 2^-} F(x) = \frac{1}{3} (2)^3 = \frac{8}{3}$
    * $\lim_{x \to 2^+} F(x) = 6(2) - \frac{1}{2} (2)^2 - \frac{22}{3} = 10 - \frac{22}{3} = \frac{8}{3}$
    * $F(2) = \frac{1}{3} (2)^3 = \frac{8}{3}$
    三者相等，故 $F(x)$ 在 $x=2$ 处**连续**。

### 四、综合题 (本大题共 2 小题, 第 19 题 10 分, 第 20 题 12 分, 共 22 分)

**19. 做一个容积为 $64\pi$ 立方米的圆形无盖容器，底面和侧面材质相同且厚度不计，问：底面半径为何值时，才能使所用材料最省？**
* **【答案】** $4$ 米
* **【解析】** 设半径为 $r$，高为 $h$。约束条件：$V = \pi r^2 h = 64\pi \Rightarrow h = \frac{64}{r^2}$。
    材料面积：$S(r) = \pi r^2 + 2\pi r h = \pi r^2 + \frac{128\pi}{r}$。
    求导：$S'(r) = 2\pi r - \frac{128\pi}{r^2}$。
    令 $S'(r) = 0$，得 $2\pi r = \frac{128\pi}{r^2} \Rightarrow r^3 = 64 \Rightarrow r = 4$。
    二阶导：$S''(r) = 2\pi + \frac{256\pi}{r^3}$。 $S''(4) = 6\pi > 0$，故 $r=4$ 时有极小值，即材料最省。

**20. 设坐标原点作曲线 $y = \ln x$ 的切线 $L$，该切线与直线 $x=1$ 及 $y = \ln x$ 围成平面图形 $D$。**
*【注：本题附有曲线、切线和区域 $D$ 的图示，请参考原试卷】*

**(1) 求切线 $L$ 的方程;**
* **【答案】** $y = \frac{1}{e} x$
* **【解析】** 设切点为 $(x_0, \ln x_0)$。斜率为 $y'|_{x=x_0} = \frac{1}{x_0}$。
    切线方程：$y - \ln x_0 = \frac{1}{x_0} (x - x_0)$。
    切线过原点 $(0,0)$：$0 - \ln x_0 = \frac{1}{x_0} (0 - x_0) = -1$，得 $\ln x_0 = 1$，即 $x_0 = e$。
    切线斜率 $k = \frac{1}{e}$。方程为 $y = \frac{1}{e} x$。

**(2) 求平面图形 $D$ 的面积。**
* **【答案】** $\frac{e^2 - 2e - 1}{2e}$ (或 $\frac{e}{2} - \frac{1}{2e} - 1$)
* **【解析】** 区域 $D$ 在 $x \in [1, e]$ 上，由切线 $y = \frac{1}{e} x$ 和曲线 $y = \ln x$ 围成。
    $$S_D = \int_1^e \left( \frac{1}{e} x - \ln x \right) dx$$
    $$S_D = \left[ \frac{1}{e} \cdot \frac{x^2}{2} \right]_1^e - [x \ln x - x]_1^e$$
    $$S_D = \left( \frac{e^2}{2e} - \frac{1}{2e} \right) - \left[ (e \ln e - e) - (1 \ln 1 - 1) \right]$$
    $$S_D = \left( \frac{e}{2} - \frac{1}{2e} \right) - (0 - (-1)) = \frac{e}{2} - \frac{1}{2e} - 1$$
    $$S_D = \frac{e^2 - 1 - 2e}{2e}$$
